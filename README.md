# DRAG-AND-DROP

### Set Up Instructrion

1. Clone The Project
```
    git clone https://github.com/manish25Coder/Drag-And-Drop.git
``` 

2. Move Into The Directory 
```
    cd drag-and-drop
```

3. Install The Dependencies 
```
    npm i
```

4. Run The Server 
```
    npm run dev
```
